# Embedded Systems Project Roadmap

Welcome to the Embedded Systems Learn and Project Repository! 🚀
This repository covers projects from beginner to advanced levels using Arduino Uno and STM32F1 boards with various sensors.

## 🛠️ Project Folders Structure
```
📂 Embedded-Systems-Projects/
├── 📝 README.md
├── 📂 Arduino_Projects/
│   ├── 01_Blink_LED/
│   ├── 02_Temperature_Monitoring/
│   ├── 03_Motion_Detector/
│   └── 04_OLED_Weather_Station/
└── 📂 STM32_Projects/
    ├── 01_Blink_LED/
    ├── 02_UART_Communication/
    ├── 03_PWM_Motor_Control/
    └── 04_ADC_Temperature_Sensor/
```

## 📘 Arduino Projects
1. **Blink LED**: Simple LED blink with `digitalWrite()`.
2. **Temperature Monitoring**: Use a DHT11 sensor to read temperature and display it on the Serial Monitor.
3. **Motion Detector**: Detect motion with a PIR sensor and trigger a buzzer.
4. **OLED Weather Station**: Display temperature and humidity on an OLED screen.

## 📘 STM32 Projects
1. **Blink LED**: Basic LED blink using HAL drivers.
2. **UART Communication**: Send and receive data over UART.
3. **PWM Motor Control**: Control a motor’s speed using PWM.
4. **ADC Temperature Sensor**: Read temperature using the internal ADC.

## 🚀 Future Project Ideas
- Smart Home Automation with relays and sensors.
- IoT Weather Station using ESP8266 with STM32.
- Data Logger with SD card and RTC module.
- Ultrasonic Distance Meter with STM32.

Contributions and suggestions are always welcome! 😊✨
